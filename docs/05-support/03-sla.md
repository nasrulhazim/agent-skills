# Support SLA — Agent Skills

> This SLA applies to the open-source community
> support provided via GitHub Issues.

## Severity Levels

| Level | Label | Description | Example |
| --- | --- | --- | --- |
| P1 | Critical | Install breaks | Deletes files |
| P2 | High | Core skill broken | Invalid code |
| P3 | Medium | Partial breakage | Missing ref |
| P4 | Low | Minor or cosmetic | Typo in desc |

## Response Targets

| Severity | First Response | Resolution |
| --- | --- | --- |
| P1 | Within 24 hours | Within 72 hours |
| P2 | Within 3 biz days | Within 2 weeks |
| P3 | Within 1 week | Next minor release |
| P4 | Best effort | Backlog |

> **Note:** These are targets, not guarantees.
> This is a solo-maintained open-source project.

## Out of Scope

The following are not covered by this SLA:

- Issues caused by unsupported Claude Code versions
- Custom modifications to skill files
- General Laravel/PHP help unrelated to the skills
- Third-party tool conflicts

## Security Vulnerabilities

Do **not** open a public GitHub Issue for security
vulnerabilities. Email the maintainer directly with
subject `[SECURITY] Agent Skills`. Response within
48 hours guaranteed.
