# Report Templates

Markdown templates for all report types generated by the log-monitor skill.

---

## Top-Level Summary (Multi-Project)

```markdown
# [Organization] - Log Analysis Summary

**Date:** YYYY-MM-DD
**Period Analyzed:** YYYY-MM-DD to YYYY-MM-DD

---

## Project-Level Summary

```markdown
# [Project Name] - Log Analysis Summary

**Period:** YYYY-MM-DD to YYYY-MM-DD
**Environment:** production at /path/to/app
**Database:** PostgreSQL / MySQL / Oracle
**Total Entries:** X (Y errors + Z warnings)

## Daily Totals

| Date | Errors | Warnings | Total | Key Event |
|---|---|---|---|---|
| YYYY-MM-DD | X | Y | Z | Brief description or — |

## Error Distribution by Category

| Category | Count | % | Status |
|---|---|---|---|
| Category name | X | Y% | ACTIVE / Likely resolved / Ongoing |

## Key Actions

1. **Action title** - Brief description
2. **Action title** - Brief description

## Links

- [Consolidated Error Types](error-type.md)
- [Consolidated Action Items](todo.md)
- Daily: [Mon DD](YYYY-MM-DD/summary.md) | ...
```

## Project-Level Error Types

```markdown
# [Project Name] - Error Types (All Dates)

## Summary

| # | Error Type | Total Count | % | Category | Status |
|---|---|---|---|---|---|
| 1 | Error name | X | Y% | Category | Active/Resolved |

---

## [Category Name]

### [Specific Error Name]

- **Total Count:** X across Y days
- **Level:** ERROR / WARNING / CRITICAL
- **Exception:** Full\Exception\ClassName
- **Message:** `Actual error message text`
- **Affected:** Who or what is impacted
- **Key Stack Files:**
  - `app/Services/SomeService.php:123`
  - `app/Http/Controllers/SomeController.php:45`
- **First Seen:** YYYY-MM-DD
- **Last Seen:** YYYY-MM-DD
- **Status:** Active / Resolved after YYYY-MM-DD / Intermittent
- **Root Cause:** Known or suspected cause
```

## Daily Error Types

```markdown
# [Project Name] - Error Types - YYYY-MM-DD

## [Category Name]

### [Specific Error]

- **Count:** X
- **Level:** ERROR / WARNING
- **Exception:** Full\Class\Name

- **Affected:** Users, endpoints, features affected

```

  - Count: X occurrences
- [ ] [Priority: P1/P2/P3/P4] Another action
  - Error: Brief error description
  - Count: X occurrences
```
